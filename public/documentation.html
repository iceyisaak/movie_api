MyFlix API Documentation

This is the API documentation. Here, developers can find relevant endpoints to interact with our the database.

List of All Endpoints

Request URL Response
GET: Fetch all movies /movies JSON: Data of all movies in the database
GET: Fetch a single movie by title /movies/:title JSON: Data of a particular movie in the database
GET: Fetch a genre by genre name /movies/genre/:name JSON: Movie genre of a particular movie
GET: Fetch a movie director by name /movies/:director/:name JSON: Data of a movie director (along with bio, birth year, and dead year)
POST: Add a new user /users JSON: Data of recently added user
POST: User adds a movie to the list of favourite /users/:username/:movie/:id Text: Showing that the user has added a new movie successfully from a list of favourite.
PUT: Update user info by username /users/:username Text: Showing that user data has been successfully updated.
DELETE: User can remove a movie from the list of favourite /users/:username/:movie/:id Text: Showing that the user has removed a movie successfully from a list of favourite
DELETE: User can deregister from the platform /users/:username Text: Showing that the user data has removed from the platform

MyFlix API Documentation

This is the API documentation. Here, developers can find relevant endpoints to interact with our the database.

List of All Endpoints

Request Method URL Response

Get all movies

Get a list of all the movies and their information available in the database of the platform.

GET /movies

JSON Object

Returns data of all movies in present the database of the platform. The details include:

  • _id
  • Title
  • Description
  • Genre
  • Actors
  • Featured 
     JSON
      
       {
         {
          "Genre": {
              "Name": "Romance",
              "Description": "Romance films are romantic......"
          },
          "Director": {
              "Name": "Nora Ephron",
              "Bio": "Nora Ephron was educated at.......",
              "Birth": "1941"
          },
          "Actors": [],
          "_id": "601efbe5d0f56932960a25a7",
          "Title": "Bewitched",
          "Description": "Thinking he can overshadow an unknown...",
          "Featured": true,
          "Bio": "Nora Ephron was an American journalist..."
         }
       }

Get a movie genre by title

Get a movie genre by the movie title along with its description.

GET /movies/:Title/genre JSON Object

Returns the movie genre of the particular movie

  • Genre Name
  • Genre Description 
     JSON
      
       {
        "Name": "Romance",
        "Description": "Romance films or romance movies are..."
       }

Get a movie director by name

Get a movie director by his/her fullname along with the biography, year of birth, and year of death (if any)

GET /movies/director/:Name JSON Object

Returns the data of the movie director along with details including:

  • Bio
  • Birth year
  • Death year (if any) 
     JSON
      
       {
        "Name": "Jonathan Demme",
        "Bio": "Robert Jonathan Demme was an American director...",
        "Birth": "1944",
        "Death": "2017"
       }

Add a new user

Add a new user to the platform. Users can enter personal details, including:

  • Username
  • Password
  • Email
  • Birthdate
Input Format: 
     
      {
       "Username": "jondoe",
       "Password": "1234",
       "Email": "jondoe@gmail.com",
       "Birthday":"11/12/1990"
      }
POST /users JSON Object

Returns the data of recently added user's information including:

  • _id
  • Username
  • Password
  • Email
  • Birthday
  • FavouriteMovies 
     JSON
      
       {
        "_id": "60228aca3dac8501d3a328d3",
        "Username": "jondoe",
        "Password": "1234",
        "Email": "jondoe@gmail.com",
        "Birthday": "1990-11-11T23:00:00.000Z",
        "FavouriteMovies": [],
        "__v": 0
       }

Update a user's information by username

Edit a platform user's personal information by username. The details can be altered simply entering by a new details.

Any details wished to be kept, must also be specified in its original form and NOT omitted.

The details include:

  • Username
  • Password
  • Email
  • Birthdate

Input Format

: 
     JSON
     
      {
       "Username": "jondoe",
       "Password": "1234",
       "Email": "jondoe@gmail.com",
       "Birthday":"11/12/1990"
      }
PUT /users/:Username JSON Object

Returns the data of recently updated user's information including:

  • _id
  • Username
  • Password
  • Email
  • Birthday
  • FavouriteMovies 
     JSON
      
       {
        "_id": "60228aca3dac8501d3a328d3",
        "Username": "jondoe",
        "Password": "1234",
        "Email": "jondoe@gmail.com",
        "Birthday": "1990-11-11T23:00:00.000Z",
        "FavouriteMovies": [],
        "__v": 0
       }

Add a movie to a user's list of favourite movie

POST /users/:Username/movies/:MovieID

JSON Object

Returns the user data in JSON format along with the MovieID being added into the 'FavouriteMovies' array

  • _id
  • Username
  • Password
  • Email
  • Birthday
  • FavouriteMovies 
     JSON
      
       {
        "FavouriteMovies": [
            "601ef184d0f56932960a259f"
        ],
        "_id": "60228aca3dac8501d3a328d3",
        "Username": "jondoe",
        "Password": "1234",
        "Email": "jondoe@gmail.com",
        "Birthday": "1990-11-11T23:00:00.000Z",
        "__v": 0
    }

Remove a user by username

Remove an existing platform user from the platform.

DELETE /users/:Username

Text

Returns a text message showing that the user data has been successfully removed from the platform.
